 Vaccination Record Console System

Project Description
This project is a Java console application that manages vaccination records. It allows registering patients, store vaccination details, and view records easily.

 Features
•	Add new patients
•	Add vaccination records
•	View all patients
•	View all vaccination records

What was used
•	Java
•	Object-Oriented Programming (OOP)
•	ArrayLists for data storage
•	Console-based input/output

Structure
•	Patient class → stores patient details
•	Vaccine class → abstract class for vaccine types
•	VaccinationRecord class → stores vaccination data
•	DataStore → stores arraylist
•	MainApp → main console application
